
# üè¶ Bankist

Welcome to Bankist, a sophisticated, pure JavaScript web application that simulates a modern online banking interface. This project is a comprehensive demonstration of advanced JavaScript concepts including efficient array manipulation, event handling, and the use of the Internationalization API for formatting currency and dates.


## ‚ú® Features

- Account Management: Calculates and displays current balance, total deposits (IN), total withdrawals (OUT), and calculated interest.

- Core Operations: Includes Transfers, Loan Requests, and Account Closure.

- Transaction Display: Shows deposits and withdrawals with dates formatted using the Internationalization API and relative time (Today, Yesterday, etc.).

- Data Sorting: Ability to sort transactions (movements) by amount.

- Sticky Navigation & Visual Effects: Uses the Intersection Observer API to implement a sticky navigation bar, lazy load images, and reveal sections as they scroll into view.

- Logout Timer: Automatic logout after a period of inactivity.
